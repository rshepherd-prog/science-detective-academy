🧠 LOGIC LAB

Now we solve mysteries differently. Instead of collecting clues, we start with rules.

FACT: The beaker has mass.

What can you conclude?

DETECTIVE SKILL UNLOCKED

DEDUCTIVE REASONING

You started with a rule and applied it to a specific case.

Rule → Evidence → Conclusion

When the rule is true and the evidence matches, the conclusion must follow.

🏆 CASE COMPLETE

Detective Report

You learned two ways scientists reason.

🔍 Inductive Reasoning

Observations → Pattern → Likely Explanation

Used when we look for patterns and make predictions.

📋 Deductive Reasoning

Rule → Specific Evidence → Certain Conclusion

Used when we apply known rules to specific situations.
